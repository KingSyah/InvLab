⚙️ Pengaturan Google Sheets
Cara menggunakan:
1. Buka script.google.com dan buka project terhubung ke spreadsheet kamu.
2. Ganti isi Code.gs dengan kode di bawah.
3. Klik Deploy → New deployment → pilih tipe Web app.
4. Set Execute as: Me dan Who has access: Anyone.
5. Salin URL deployment dan paste di kolom di atas, lalu klik Simpan.
6. Klik 🔌 Test Koneksi untuk memastikan berhasil.
